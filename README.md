# dotfiles

개발 환경 설정 파일 모음. `install.sh`로 심볼릭 링크를 생성한다.

## 설치

```bash
git clone <repo-url> ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## 구성

| 도구 | 파일 |
|------|------|
| nvim | `nvim/` |
| tmux | `tmux/.tmux.conf` |
| zsh | `zsh/.zshrc`, `zsh/.zprofile` |
| git | `git/.gitconfig` |
| gh | `gh/config.yml` |
| claude | `claude/CLAUDE.md`, `claude/settings.json` |
| vscode | `vscode/settings.json`, `vscode/keybindings.json` |
