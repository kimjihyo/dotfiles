⏺ # Global Preferences

  ## 언어

  - 항상 한국어로 답변

  ## 패키지 매니저

  - pnpm을 우선해서 사용


  ## 커밋

  - 한국어로 자세하게 작성
  - 왜 이 기능추가/변경을 하게 되었는지 맥락을 포함
  - 이 변경으로 생긴 결과 / 기대되는 결과를 포함
  - 성능 개선이 이루어졌다면 얼만큼의 성능 개선이 이루어졌는지 정량적인 지표도 포함
  - 접두사: feat:, fix:, chore:, refactor:, docs:

  ## 커뮤니케이션

  - 간결하게 답변
  - 이모지 사용하지 않음
